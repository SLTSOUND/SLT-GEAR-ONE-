# SLT GEAR ONE - Subwoofer Testing Interface

A professional web-based testing interface for analyzing subwoofer performance. This application provides real-time frequency response analysis, waveform visualization, and performance metrics designed specifically for subwoofer testing.

![SLT GEAR ONE Interface](https://via.placeholder.com/800x450/121a36/ffffff?text=SLT+GEAR+ONE)

## Features

- **Real-time Analysis**: Visualize frequency response, waveform, and spectrogram data in real-time
- **Multiple Test Types**: Run frequency sweeps, sine wave tests, pink noise, and pulse response tests
- **Performance Metrics**: Measure and analyze key subwoofer performance indicators:
  - Peak SPL (Sound Pressure Level)
  - Resonant Frequency
  - Power Consumption
  - Distortion Percentage
- **Automated Analysis**: Get instant feedback on subwoofer performance characteristics
- **Adjustable Parameters**: Customize frequency range, amplitude, and test duration
- **Results Export**: Save test results for later analysis or comparison

## Technical Specifications

- **Frequency Range**: 20Hz - 200Hz
- **Amplitude Range**: -20dB to +6dB
- **Sample Rate Options**: 44.1kHz, 48kHz, 96kHz
- **Data Resolution**: 24-bit

## Getting Started

1. Open `index.html` in a modern web browser (Chrome, Firefox, Edge recommended)
2. Allow microphone access when prompted for input device detection
3. Configure your audio input and output devices in Settings
4. Set desired test parameters
5. Click "START TEST" to begin analysis

## System Requirements

- Modern web browser with Web Audio API support
- Microphone input for measurements
- Audio output device connected to subwoofer

## Usage Notes

- For optimal results, place the microphone at the listening position
- Ensure ambient noise is minimized during testing
- Begin with lower amplitude settings and gradually increase if needed
